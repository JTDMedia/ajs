Android.js

Bring the power of Node.js to your Android device

What is Android.js?

A Powerful Framework for Developers

Android.js is a revolutionary framework that lets you run Node.js on your Android device. Built for developers with experience in Node.js, it makes it easier than ever to manage server-side JavaScript directly on Android.

Official Code Editor

To enhance your development experience, Android.js offers an official code editor for seamless integration. Edit, debug, and run your projects without leaving your mobile device.

How to Install Android.js

Quick Installation Guide

  1. Download the APK from the GitHub Releases.
  2. Install the APK on your Android device.
  3. Open the app and grant all necessary permissions.
  4. Select the folder containing your Node.js project.
  5. Use the console to manage your project:
    • Replace node with droid.
    • Replace npm with apm.

Key Features

  • Run Node.js applications on Android devices.
  • Lightweight and efficient framework.
  • Compatible with Android 11 and higher.
  • Requires >4GB RAM and 1GB storage.
  • Fully customizable console and file access.
